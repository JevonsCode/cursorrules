## 技术栈规范

### 前端开发

1. [Mandatory] 语言选择优先级:
   - TypeScript (TS) > JavaScript (ES6+)
   - 框架版本:
     • Vue 最新版本
     • React 最新版本

2. [Recommended] 包管理器:
   - 优先使用 pnpm 最新版本

### 后端开发

1. [Mandatory] 基础架构:
   - Node.js 最新版本
   - MongoDB 最新版本

2. [Recommended] 框架选择:
   - Node 相关框架

### 样式规范

1. [Mandatory] 预处理器:
   - SCSS/SASS 最新版本

2. [Restricted] 禁止属性:
   - gap (flex)
   - 非标准厂商前缀

### 工具链配置

1. [Mandatory] 依赖管理:
   - pnpm 最新版本

2. [Recommended] 构建工具:
   - Vite 最新版本
